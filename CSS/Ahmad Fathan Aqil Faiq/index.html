Berbagi Informasi Seputar Diri Kalian

Saya ingin tahu seputar tentang diri kalian, tidak lupa saya juga akan membagikan seputar diri saya sendiri. Apa sih maksudnya? Sebenarnya ini hanya sekedar untuk tahu apa sih yang kalian suka seperti game favorit kalian atau makanan kesukaan kalian atau pun apa hobi kalian di kala senggang, hanya seputar itu saja. Tujuannya? Ya siapa tahu kalau ternyata game favorit kita sama, bisa maen bareng. Kalau makanan favorit sama bisa kapan kapan nongkrong bareng. Kalau hobi kita sama bisa kapan kapan hangout bareng atau ga sharing - sharing biasa, ya hanya seputar itu lah. Maka dari itu, jika kalian tidak keberatan kalian bisa lihat ini sebentar dan kalau bisa untuk mengisinya juga.

Sebelumnya, saya akan memberitahu tentang diri saya terlebih tahulu, ya dihitung hitung sebagai pengenalan awal.
