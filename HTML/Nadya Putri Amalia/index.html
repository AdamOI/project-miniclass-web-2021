OKAY LANJUT...

Jadi HTML adalah sebuah bahasa markah yang digunakan untuk membuat sebuah halaman web, menampilkan berbagai informasi di dalam sebuah penjelajah web Internet dan pemformatan hiperteks sederhana. HTML adalah sebuah standar yang digunakan secara luas untuk menampilkan halaman web.
Kita belajar dimulai dari ;

  1. Bagaimana mengatur formating
  2. Menambahkan Link untuk dapat mengakses website lain
  3. Menambahkan Media (Gambar, Audio, Video)
  4. Membuat list seperti ini
  5. Membuat tabel
  6. Membuat form untuk login
  7. Dan yang terakhir yaitu Grouping
